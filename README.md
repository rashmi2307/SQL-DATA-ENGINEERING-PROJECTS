# SQL Data Engineering Projects

Hands-on projects completed while learning and applying core data engineering concepts including SQL analytics, data warehousing, dimensional modeling, ETL pipelines, and data mart design.

# Projects

## [1_EDA](./PROJECT_1_EDA/) - Exploratory Data Analysis

![EDA Project Overview](./Images/1_1_Project1_EDA.png)

SQL-driven analysis of remote Data Engineer job market trends, focusing on skill demand, salary insights, and identifying the most valuable skills based on both compensation and market demand.

**Skills**: Complex joins, aggregations, analytical functions, statistical analysis, data exploration

## [2_WH_Mart_Build](./PROJECT_2_WH_Mart_Build/) - Data Warehouse & Data Mart Build

![Data Pipeline Architecture](./Images/1_2_Project2_Data_Pipeline.png)

End-to-end ETL pipeline that transforms raw job posting data into a normalized star schema data warehouse and multiple analytical data marts optimized for reporting and business analysis.
